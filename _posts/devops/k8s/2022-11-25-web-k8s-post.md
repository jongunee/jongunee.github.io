## NFS 서버 설치

**root 계정으로 접속**

```cmd
sudo -i
```


**NFS 패키지 프로그램 설치**

```cmd
apt install nfs-common nfs-kernel-server portmap
```

안된다면 업데이트 수행

**공유 폴더 생성**

```cmd
mkdir /home/share/nfs -p
```

**폴더 권한 부여**

```cmd
chmod -R 777 /home/share/nfs
```

**공유폴더 연결 호스트 및 권한 설정**

```cmd
vim /etc/exports
```

아래 라인 추가 (예시)

```
/home/share/nfs       192.168.72.58(rw,sync)
```

**디렉토리 목록 리플래쉬**

```cmd
exportfs -r
```
