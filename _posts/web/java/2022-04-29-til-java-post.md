# JAVA 디버깅 방법

1. 디버깅 버튼 클릭
2. Break point 지정 - 라인 번호 옆에 더블클릭
3. 디버깅
    - F5 함수 안으로 들어가기
    - F6 한 라인씩 진행
    - F7 원래대로 돌아가기
    - F8 그 다음 Break point까지 이동
    - 빨간색 중단 버튼
